{# Cmp-card diff body. Reads cur + cmp text from sibling DOM and renders a word-level inline diff. Refreshes when: * the cmp `diff` toggle flips on (x-init runs) * the page-level mode flips (prompt ↔ output) * a run's output swaps (pendingRunSwap) — but ONLY in output mode; in prompt mode the text is the two saved bodies, which a clip click / run (pendingRunSwap) never changes, so re-running the O(n·m) word diff then would be wasted work. The `mode === 'output' && …` gate makes pendingRunSwap a dependency only while output is shown. * the cur draft is saved (savedTick bumps) — without this, saving a changed prompt left the diff showing the pre-save text. Deps are read off `$store.studio` (reactive). They were previously read as `$root.*`, but inside this cmpDiff x-data `$root` is the diff div itself, so `$root.activeVersionId` etc. were undefined element-property reads that created no reactive dependency — the x-effect never re-fired on save. #}
